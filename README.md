# NEURAL-MESH
"Autonomous AI system featuring Hyperdimensional Computing (HDV), multi-hop reasoning, and a verifier-first architecture."
INFINITY v4.0 is an autonomous "Living Knowledge Organism" built in C++. Unlike traditional LLMs, it focuses on verifiable symbolic reasoning and local memory management. It uses Hyperdimensional Vector (HDV) encoding to represent concepts and a Knowledge Mesh to store relationships (triples).
Key Features:
Autonomous Learning: Integrated Wikipedia crawler using libcurl to grow its own memory.
Hyperdimensional Computing: Uses 8192-dimensional vectors for semantic concept encoding and LSH (Locality Sensitive Hashing) for fast retrieval.
Strict Verifier Oath: A logic-driven system that refuses to make unverified claims, flagging contradictions and knowledge gaps.
Dual-Layer Memory: Full-text search (FTS5) for sentences and a Triple-based Knowledge Mesh for relational logic.
Reasoning Cortex: Supports multi-hop pathfinding, transitive inference, and causal reasoning.
Autonomic Medulla: Real-time system monitoring (CPU/RAM/Load) that adjusts the organism’s "hunger" and "curiosity" levels.
